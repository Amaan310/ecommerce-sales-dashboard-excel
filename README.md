# E-Commerce Analytics Dashboard (Excel)

## ðŸ” Project Overview
This project presents a multi-channel e-commerce analytics dashboard built using **Microsoft Excel** to analyze customer behavior, sales performance, and order trends across multiple online platforms such as **Amazon, Flipkart, Myntra, Ajio, Meesho**, and others.

The dashboard transforms **31,000+ raw transactional records** into actionable business insights, enabling data-driven decision-making for marketing, operations, and inventory planning.

---

## ðŸŽ¯ Business Questions Answered
The dashboard was specifically designed to address the following key performance indicators:

* **Sales Trends:** How do sales and orders trend month-wise?
* **Demographics:** Which gender contributes more to revenue?
* **Order Status:** What percentage of orders are delivered, returned, refunded, or cancelled?
* **Geography:** Which states generate the highest sales?
* **Behavioral Analysis:** How does age group vs. gender affect purchasing behavior?
* **Channel Performance:** Which sales channel performs best in terms of order volume?

## ðŸ›  Tools & Techniques Used
* **Microsoft Excel:** Core platform for data processing and visualization.

* **Analytics Concepts:** Customer segmentation, sales performance analysis, and KPI-driven storytelling.

## ðŸ“‚ Dataset Description
The dataset consists of **31,049 orders** with the following key fields:

| Category | Key Fields |
| :--- | :--- |
| **Identity** | Order ID, Customer Gender |
| **Demographics** | Customer Age, Age Group, City, State, Pincode |
| **Product** | Category, Size |
| **Logistics** | Sales Channel, Order Status, Date |
| **Financial** | Order Amount |

> **Note:** The raw data was cleaned and standardized to ensure consistency across cities, states, sizes, and order statuses.

---

## ðŸ“ˆ Key Insights & Findings
* **Gender Contribution:** Women contribute **~64%** of total sales, indicating a strong female customer base.
* **Operational Success:** **92%** of orders are successfully delivered, reflecting high efficiency.
* **Top Regions:** Maharashtra, Karnataka, and Uttar Pradesh are the top-performing states by revenue.
* **Platform Dominance:** Amazon and Myntra together contribute nearly **60%** of total orders.
* **Customer Segment:** Adult customers dominate purchases, especially among women.
* **Returns:** Returns and refunds remain **below 6%**, which is healthy for e-commerce operations.
